## Hardware Used
- TB6612FNG (Motor Driver)
- ESP32
- QTR 8RC Array
- DC Motor (Any)

## Circuit
1. TB6612FNG
   - GND (All three): ESP32 GND
   - VCC: ESP32 3.3V
   - AIN1: GPIO13
   - AIN2: GPIO14
   - PWMA: GPIO25
   - PWMB: GPIO26
   - BIN1: GPIO32
   - BIN2: GPIO33
   - VM: Battery (+)
2. QTR 8RC Array
   - None for the moment
